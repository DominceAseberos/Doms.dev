## ⚙️ High-Level Engineering Skills

GanapPH demonstrates a strong grasp of modern React ecosystem tools and conceptualizing AI-driven products from scratch.

### 1. Modern React Architecture
Engineered the application using Vite and React, leveraging TanStack Router for complex, type-safe navigation flows. Integrated React Query for efficient data fetching, caching, and state management, ensuring the UI remains highly responsive during AI data generation.

### 2. Component-Driven UI Design
Implemented a scalable design system using Tailwind CSS and Radix UI primitives. This approach ensured that complex UI components (like Accordions, Dialogs, and Context Menus) were fully accessible, unstyled by default, and heavily customized to match the brand's aesthetic without compromising performance.

### 3. Product Strategy & Pitching
Developed GanapPH as a Technopreneurship project, requiring not just technical execution, but also product strategy. The platform was designed to solve a real-world problem (career mapping) by defining clear user stories and translating them into an actionable MVP (Minimum Viable Product).
