# 🎯 GanapPH — AI-Powered Career Guide

**GanapPH** is an intelligent web platform conceived as a Technopreneurship pitch project. It serves as a personalized guide for job seekers, leveraging AI to generate step-by-step career pathways and highly targeted job recommendations based on user profiles and skills.

### 🧠 Concept & Purpose

The platform aims to bridge the gap between education and employment by providing actionable, AI-driven insights. Instead of simply listing jobs, GanapPH analyzes a user's background and tells them *exactly what to do next*—whether that means learning a new skill, optimizing their resume, or applying to specific entry-level roles.

### 🛠 Tech Stack

- **Frontend**: Built with **React** and **Vite** for blazing-fast performance.
- **Routing & State**: Utilizes **TanStack Router** for type-safe navigation and **React Query** for robust asynchronous state management.
- **UI Architecture**: Designed with **Tailwind CSS** and heavily integrated with **Radix UI** primitives for accessible, customizable components.

### ✨ Key Features

- **Personalized Step-by-Step Guides**: Generates a custom roadmap tailored to the user's specific career goals.
- **Intelligent Job Matching**: Recommends relevant job openings by matching user details with current market demands.
- **Modern, Accessible UI**: A highly responsive interface built with modern React patterns, ensuring a smooth experience across all devices.

GanapPH represents a forward-thinking approach to career development, blending modern frontend engineering with AI-driven user experiences.
