## ⚙️ High-Level Engineering Skills

Building Baylora required robust mobile state management and seamless integration with Backend-as-a-Service (BaaS) architecture.

### 1. Advanced Flutter State Management
Implemented a highly reactive UI using **Riverpod**, ensuring complex state—like real-time bidding, user authentication status, and active item listings—remains consistently synchronized across different screens without performance bottlenecks.

### 2. Relational Database Design
Architected a normalized PostgreSQL database schema within Supabase. This involved linking dynamic entities (`items`, `profiles`, and `offers`) using Foreign Keys to efficiently handle complex multi-type transactions (Trade/Cash/Mix) and maintain data integrity during bidding processes.

### 3. Serverless Backend Integration
Effectively utilized the entire Supabase ecosystem. Leveraged Supabase Auth for secure profile management and utilized Supabase Storage to handle performant, scalable media uploads for marketplace item images.
