# Doms.dev - Portfolio v2

Personal portfolio website showcasing projects, skills, and professional experience.

## Tech Stack

| Category | Technologies |
|----------|-------------|
| **Frontend** | React 18, Vite, TailwindCSS, GSAP |
| **Backend** | Supabase (PostgreSQL, Auth, Storage, Edge Functions) |
| **State** | Zustand, TanStack Query |
| **AI** | Gemini, OpenRouter (via Edge Functions) |

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

> AI API keys are stored securely in Supabase Edge Function secrets.
